modelscope-jobs

Reusable workflow scripts for data generation, evaluation, and artifact publishing on ModelScope Hub.

Quick Start

export MODELSCOPE_API_TOKEN=<token>
uv run scripts/generate-responses.py \
  --source-dataset your-org/prompts \
  --output-dataset your-org/generated \
  --model Qwen/Qwen2.5-7B-Instruct
