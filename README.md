# ModelScope Skills

ModelScope Skills is an independent community project with reusable agent skills for dataset engineering, model training, evaluation, tracking, reporting, and workflow automation for ModelScope workflows.

The repository follows the open Agent Skills format: each skill is a self-contained directory with a `SKILL.md` entrypoint plus scripts, references, and templates.

## Installation

### Claude Code

```text
/plugin marketplace add hyf020908/modelscope-skills
/plugin install <skill-name>@hyf020908/modelscope-skills
```

Example:

```text
/plugin install modelscope-cli@hyf020908/modelscope-skills
```

### Codex

Copy or symlink selected skill folders from `skills/` into your standard `.codex/skills` location.

### Gemini CLI

```bash
gemini extensions install https://github.com/hyf020908/modelscope-skills.git --consent
```

### Cursor

Use `.cursor-plugin/plugin.json` and `.mcp.json` from this repository.

## MCP Setup

This repository ships MCP client config examples aligned with the official `modelscope-mcp-server` package.

1. Install and expose token in your shell:

```bash
export MODELSCOPE_API_TOKEN="<your-token>"
```

2. Use stdio MCP mode (already encoded in `.mcp.json` and `gemini-extension.json`):

```json
{
  "mcpServers": {
    "modelscope-mcp": {
      "command": "uvx",
      "args": ["modelscope-mcp-server"]
    }
  }
}
```

## Contributing

1. Duplicate an existing skill directory under `skills/`.
2. Update the new directory's `SKILL.md` frontmatter (`name`, `description`).
3. Add scripts/references/templates needed by the skill.
4. Update `.claude-plugin/marketplace.json`.
5. Run `./scripts/publish.sh` and commit generated files.
